# Mod Engine (Mindustry V8)

**Mod Engine** - Rhino/JS панель управления и runtime для **Mindustry V8** (minGameVersion **154+**, проверка на **159.x**).

В игре открывается интерфейс **NexusOS**: инструменты sandbox, флот-майнинг, inspector контента, оверлеи радиусов и JS-консоль.

---

## Установка

1. Скопируйте папку мода (корень репозитория) в:
   - ПК: `Mindustry/mods/`
   - Android: `Mindustry/mods/`
2. Перезапустите Mindustry.
3. Включите **Mod Engine** в списке модов.
4. При **первом запуске** появится приветственное окно (**EN/RU**) с важными советами.

---

## Как открыть меню

Кнопка в HUD: **Mod Engine** (иконка настроек рядом с мобильными кнопками / status table).

Режимы навигации:

| Режим | Смысл |
|-------|--------|
| **ALL** | Все вкладки |
| **USUAL** | Обычная игра |
| **SANDBOX** | Читы / оверрайды |

## Функции

### Home
- Сектор и оценка угрозы
- **Поток предметов ядра** (количество + дельта/сек)
- Баланс энергосети

### Waves
- Принудительная волна / auto-wave
- Превью состава волны
- Скорость симуляции

### World
- Время суток и ambient light
- Сила ветра
- Сброс погоды
- Захват сектора / перенос команд (кампания)

### Items
- Инъекция предметов в ядро
- Fill / clear склада ядра
- Фильтры base / modded

### Units
- Спавн юнитов (союзник/враг, количество)
- Фильтры планеты / модов
- Диалоги действий (HP, команда и т.д.)

### Player
- Макс. HP, скорость, прыжок, mine speed, реген
- Статусы, heal / ammo / self-destruct

### Weapon
- Урон / дальность / разброс оружия юнитов
- Instant reload **только для своей команды**
- Баффы турелей (reload / range / damage / spread)

### Mining
- Глобальные множители скорости добычи
- **Карточки флота**: назначение одной/нескольких руд типу юнита
- Через CommandAI stances (несколько руд одновременно)

### Inspector
- Браузер блоков / юнитов / предметов
- Каталог модов
- **Пагинация** — если лагает, снизьте **PAGE SIZE** до **20–40**

### Builds
- Instant build / godmode структур
- Зелёное выделение области + массовые действия

### Radius
- Оверлей дальности турелей
- Оверлей дальности оружия / майнинга юнитов
- Стиль force-shields (без наложения, свой FBO)

### Console
- Интерактивный Rhino JS
- Алиасы: `player`, `state`, `world`, `logic`, `items`
- Очистка лога / экспорт trace

### Hotkeys / Links
- Заготовки хоткеев
- Ссылки / support stubs

---

## Советы по производительности

1. **Лагает Inspector** → PAGE SIZE **20–40**.  
2. **Много модов** → фильтры модов, не грузите всё сразу.  
3. **Radius** → выключайте на слабых устройствах в больших боях.  
4. **Console** → не гоняйте тяжёлые циклы каждый кадр.
